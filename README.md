<p> 1: Linux / Mac? </p>

```bash
sudo apt update
sudo apt install libvulkan-dev vulkan-headers vulkan-tools glslang-tools
sudo apt install build-essential cmake libglm-dev libglfw3-dev
mkdir build
cd build
cmake ..
make
./VulkanTest
```
